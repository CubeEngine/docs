# CubeEngine - Writer

Edit signs and books!

## Commands

#### edit

Edit a sign or a signed book

Usage: `edit [1 <1st line>] [2 <2nd line>] [3 <3rd line>] [4 <4th line>]`

Alias:
`rewrite`

Permission: `cubeengine.writer.command.edit.use`

## Additional Permissions

 - ##### `cubeengine.writer`
   Base Permission for writer
