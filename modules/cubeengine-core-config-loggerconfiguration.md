# LoggerConfiguration

```yaml
# Logging into Console
# ALL > TRACE > DEBUG > INFO > WARN > ERROR > NONE
console-level: INFO

# Logging to the main log file
# ALL > DEBUG > INFO > WARN > ERROR > NONE
file-level: INFO

# Zip all old logs to zip archives
archive-logs: true

# Whether to log commands executed by players.
log-commands: false

```
