## sudo ##
The **/sudo** command allows forcing a player to chat or execute a command

#### Usage ####
**/sudo** <player\> <message\>

#### Example ####
**/sudo** DeadPlayer /suicude : Lets DeadPlayer execute /suicide  
The command will not work if the player does not have the permission for it!  
**/sudo** TalkyPlayer I love Cookies : Lets TalkyPlayer chat "I love Cookies"

#### Permissions ####
***cubeengine.basics.command.sudo*** allows using the command
