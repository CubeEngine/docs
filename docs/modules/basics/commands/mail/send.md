## Send ##
**/mail send** : Send mail to other players.

#### Usage ####
**/mail send** <name\> <message\>

#### Example ####
Admin42 uses **/mail send Owner22** Hello!
Owner22 will receive message "Hello!"

#### Alias Commands ####
sendmail 

#### Permissions ####
***cubeengine.basics.mail.send*** allows using the command
